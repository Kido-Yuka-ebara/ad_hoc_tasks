# 成果物0-1: フォーク フロー 仕様書

## 1. 製品概要

| 項目 | 内容 |
| :--- | :--- |
| **製品名** | フォーク フロー（FORK FLOW） |
| **開発元** | NX総合研究所（ロジタンフォークに続く新サービス） |
| **コンセプト** | フォークリフトDX：スマホ一つで稼働と安全性を可視化する低コスト・簡易導入サービス |
| **リリース時期** | 2026年1月予定 |

## 2. 対象環境と導入条件

| 項目 | 内容 |
| :--- | :--- |
| **対象現場** | 倉庫や工場、特にフォークリフトを保有する企業 |
| **導入の目的** | 適正台数かつ合理的な運用の検討、安全性の確保、安価で簡単な導入 |
| **計測機材** | スマートフォン（専用アタッチメントに取り付け）。機材一式はレンタル提供 |
| **設置方法** | 専用アタッチメントをバックレストに取り付け。電気工事士などの特別な資格は不要 |
| **導入期間** | 利用申し込みから最短1週間、標準で約2週間 |

## 3. 主要機能と提供価値

| 機能カテゴリ | 機能概要 | 提供価値 / 解決する課題 |
| :--- | :--- | :--- |
| **稼働状況判別** | スマートフォンの各種センサー（加速度、角速度、傾き等）と独自アルゴリズムにより稼働状況を高精度で自動判別 | 従来把握が困難だったフォークリフトの稼働状況を可視化 |
| **詳細稼働データ** | 荷役時間（リフトの上下等）、完全停止時間、アイドリング、走行時間を取得 | 稼働状況をグラフ化し、適正台数や運用方法の検討を可能に |
| **積載・から走行判別** | 走行時間のうち、積載走行時間とから走行時間をそれぞれ判別 | から走行が多い傾向を把握し、動線改善の根拠データを取得 |
| **安全運転評価** | 急発進や急停止といった危険運転の回数を取得 | 危険運転の頻度を定量把握し、オペレーターの安全運転意識改革や安全教育に活用 |
| **データ出力** | 計測データを専用WebサイトからCSV形式でダウンロード提供 | 稼働状況や安全データを自由に分析し、業務改善に活用 |

## 4. できないこと（制約事項/ロジタンフォークとの棲み分け）

フォークフローは「低コスト・簡易導入」を追求するため、ロジタンフォークなどの高精度システムと比較して、意図的に機能が限定されています。

| 制約事項 | 内容 |
| :--- | :--- |
| **リアルタイム性** | リアルタイムでのデータ送信やWeb上での監視は非対応（あえて落とした仕様） |
| **動線（位置情報）** | フォークリフトの位置情報や動線を詳細に把握することは不可。動線詳細が必要な場合は「ロジタンフォーク」を推奨 |
| **グラフ・BI機能** | リリース時点では、Webサイト上でグラフ表示やダッシュボード機能、BIツール機能は未搭載。データはCSVまたはExcelマクロ形式で提供 |
| **計測の粒度** | 作業者の動作や姿勢といったミクロな計測は、ワークプローブシステムの領域でありフォークフローの主機能ではない |
| **前提条件** | 計測開始と終了時に、オペレーターがスマートフォンのアプリでボタンを押す作業が必要 |

## 5. 利用料金

| 項目 | 内容 |
| :--- | :--- |
| **基本料金** | 月額 100,000円 |
| **従量課金** | フォークリフト1台あたり 月額 20,000円 |

## 6. 導入効果の総括

フォークフローは、**高精度システムと簡易システムの中間**に位置するソリューションです。

- ロジタンフォークの高精度（位置情報、リアルタイム監視）を意図的に削減
- スマホ+アタッチメントによる**低コスト・短期導入**を実現
- 稼働状況と安全性の**基礎データ**を取得
- CSVデータによる**柔軟な分析**が可能

**適用場面:** 初めてのフォークリフトDX、適正台数の検討、安全運転の定量化
