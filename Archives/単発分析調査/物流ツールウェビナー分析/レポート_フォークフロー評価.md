# フォークフロー 評価レポート

## 1. ツール概要

**製品:** フォーク フロー（スマホ型フォークリフト稼働監視）  
**目的:** 稼働状況・安全性の可視化  
**料金:** 基本月額100,000円 + 1台20,000円/月（10台で月額300,000円）

## 2. 主な機能と効果

| 機能 | 効果 |
| :--- | :--- |
| 稼働状況自動判別 | 荷役・停止・走行時間を可視化 |
| 積載/から走行判別 | から走行の傾向把握 |
| 急発進・急停止検知 | 危険運転の定量化 |
| CSVデータ出力 | 自由な分析が可能 |

## 3. 制約事項

- リアルタイム監視不可
- 位置情報・動線把握不可
- BI機能なし（CSV提供のみ）
- 計測開始/終了の手動操作必要

## 4. 有用性評価

| 評価項目 | 判定 | 理由 |
| :--- | :--- | :--- |
| **適用現場** | △ | ルール整備済み現場のみ |
| **効果** | × | 稼働取得自体に意味なし |
| **データ連携** | ○ | CSV出力で分析可能 |
| **汎用性** | × | 台数適正化の優先順位づけ程度 |
| **差別化** | △ | 安全面のみ有効 |

## 5. 総合判断

**判定: 導入推奨しない（短期お試しは可）**

### 理由

1. **本質的改善なし:** 製造ラインのルール整備が最優先
2. **稼働データの意義不明:** 非稼働多=削減可能ではない
3. **即効性なし:** 導入しても改善効果は期待薄
4. **永続利用価値低:** 長期利用のメリットが見出せない

### 唯一のメリット

- **安全施策:** 人車分離等の説得材料として活用可

### 導入前の必須確認事項

- 製造ラインの後工程引き取りルールは整備済みか？
- 本質的な改善順序を守れるか？
- 短期お試し（安全データ取得）と割り切れるか？
