# 成果物0-1: ロジタン デジボ 仕様書

## 1. 製品概要

| 項目 | 内容 |
| :--- | :--- |
| **製品名** | ロジタン デジボ（Logitan Dejibo） |
| **コンセプト** | クラウド型のデジタルホワイトボード。従来のホワイトボードの良さ（一覧性、簡単単純な操作）を維持し、デジタルの利点（データ蓄積、自動記録）を融合 |
| **開発目的** | ネームプレート準備や日々のメンテナンス負担、配置情報の記録不足を解決し、現場DXの第一歩とする |
| **分類** | ロジタンシリーズのアドオン製品（時間計測用デジタルボードツール） |

## 2. 対応環境と前提条件

| 項目 | 内容 |
| :--- | :--- |
| **対象現場** | ホワイトボードとネームプレートで人員配置管理を行う倉庫・工場 |
| **動作環境** | Webブラウザー（Microsoft Edge、Google Chrome等）。アプリインストール不要 |
| **必要機材** | 既存のPCとモニター（特別なシステム機材の調達不要） |
| **グローバル対応** | 英語環境対応済み。ブラウザを英語に切り替えれば海外拠点でも即利用可能 |

## 3. 主要機能と課題解決インパクト

| 機能カテゴリ | 解決レベル | 機能概要 | 課題解決/導入効果 |
| :--- | :--- | :--- | :--- |
| **ネームプレート管理** | **E：排除** | 翌日出勤者リストから出勤者のみのデジタルネームプレートを自動表示 | 200名規模で6時間かかっていた準備作業が実質ゼロに |
| **スキル管理** | **E：排除** | 過去の作業実績に基づき、保有スキルと最終作業日を自動表示・更新 | 赤いシールを貼る手動更新作業が不要、情報の信頼性向上 |
| **データ記録** | **C：結合** | マグネット移動時に作業計測が自動開始、ロジタンWebにリアルタイム連携・蓄積 | 配置管理と作業時間計測が結合、実績管理の限界を克服 |
| **配置判断** | **C/S：結合/簡素化** | 配置エリア（座標）事前設定により、マグネット移動で作業種類を自動判別 | シフト情報や特別スキルを一目で表示、リーダーの判断を簡素化 |
| **効率化** | **S：簡素化** | 範囲選択による複数人の同時移動、よく使う配置を呼び出すパターン機能 | 物量波動に素早く対応、配置変更作業を簡素化 |
| **分析** | **C：結合** | 蓄積データはロジタンWebで分析可能（KPI管理、収支管理、多能工化率等） | リーダーは雑務から解放され、本来のマネジメント業務に集中 |

## 4. できないこと（制約事項）

| 項目 | 制約事項 | 補足事項 |
| :--- | :--- | :--- |
| **単独利用不可** | デジボはロジタンの追加アドオンであり、デジボ単独で使用不可 | ロジタン基本料金（年間24,000円〜）とセットで導入必須 |
| **BI機能非搭載** | グラフ表示やダッシュボード機能、BIツール機能は未搭載 | データ分析はロジタンWebに蓄積されたデータをExcelツールや既存機能で実施 |
| **マスターデータ入力** | ネームプレートに表示される人物情報は、利用前に別途登録・入力が必要 | CSV一括登録の可否については未確認 |
| **動作計測** | 人員の配置や作業の可視化（マクロ計測）に特化。作業者の動作・姿勢（ミクロ計測）は非対応 | ワークプローブシステムによる作業自動判別機能は将来的な機能として開発中 |

## 5. 導入効果の総括

デジボの導入は、**紙の配置図を共有可能なデジタルマップに置き換える**ようなものです。

- 毎日繰り返される準備作業（ネームプレート作成、配置記録）をデジタル化で**排除**
- 配置変更という操作を「時間計測」「スキル更新」と**結合**
- 現場の管理を劇的に**簡素化**
- データ分析の基盤を構築

**ECRS評価:** E（排除） > C（結合） > S（簡素化）の順で効果を発揮
