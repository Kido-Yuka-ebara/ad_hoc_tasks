## 業務成熟度マトリクス

### 製造ライン人員配置管理業務

**業務KPI（全レベル共通の改善指標）:**

- **配置計画作成時間**: 1日分の配置計画を作成するのに要する時間（分）
- **配置変更対応時間**: 急な欠勤・物量変動への配置変更対応に要する時間（分）
- **スキルマッチング精度**: 作業者のスキルと配置工程の適合率（%）

**業務成熟度マトリクス:**

| レベル | 状態定義 | 業務の状態 | 定常施策・投資 | 非定常施策・投資 |
|:---:|:---|:---|:---|:---|
| **Level 0** | **プロセス未体系化**<br>業務が属人化し、標準的な手順が存在しない状態 | リーダーの経験と勘による口頭指示。配置記録なし。スキル情報は個人の記憶に依存 | - リーダーの経験と知識<br>- 口頭コミュニケーション | - ✓ 業務プロセスの可視化・文書化<br>- ✓ 配置ルールの暗黙知の抽出 |
| **Level 1** | **プロセス体系化**<br>業務の流れや手順が文書化され、体系的に整理されている状態 | 配置手順の文書化。配置図の作成。配置ルールの明文化 | - 配置手順書<br>- 配置図テンプレート（紙）<br>- 配置ルール文書 | - ✓ 標準作業手順書（SOP）の作成<br>- 作業者スキル一覧表の整備 |
| **Level 2** | **手順標準化**<br>標準作業手順書（SOP）が整備され、誰が実施しても同じ手順で作業できる状態 | SOPに基づく配置計画の実施。作業者リストとスキルの電子管理。配置状況の物理的可視化 | - 配置SOP<br>- 電子管理表（Excel等）<br>- 配置可視化ツール（ホワイトボード+ネームプレート等）<br>- スキル一覧表 | - 手順遵守の仕組み構築<br>- 逸脱時の記録・報告ルール整備 |
| **Level 3** | **手順遵守**<br>標準化された手順が確実に守られ、逸脱管理の仕組みがある状態 | SOP通りの配置の徹底。配置変更時の必須記録・報告。定期的な配置実績の振り返り | - 配置変更記録フォーマット<br>- 定期レビュー会議体<br>- 逸脱管理台帳 | - ✓ デジタル配置管理ツール導入<br>- ✓ リアルタイム記録の仕組み構築 |
| **Level 4** | **可視化**<br>業務の実績やパフォーマンスがリアルタイムで可視化され、現状把握ができる状態 | 配置変更の必須即時記録。スキル情報の即時更新。配置状況の常時最新状態での可視化 | - ✓ デジタル配置管理ツール（デジボ等）<br>- ✓ 基幹システム（ロジタン等）<br>- 表示用モニター<br>- 作業者マスターデータ<br>- ネットワーク環境<br>- PC | - ✓ データ分析体制の構築<br>- ✓ 改善PDCAサイクルの確立 |
| **Level 5** | **最適化**<br>データに基づいた継続的な改善活動が行われ、業務効率が向上し続ける状態 | 蓄積データに基づく配置パターンの最適化。計画的な多能工化の推進。KPI管理による継続改善 | - ✓ データ分析ツール<br>- ✓ データ分析スキル・知識<br>- 改善会議体<br>- ✓ KPIダッシュボード<br>- 多能工化推進計画<br>- 改善文化 | - ✓ 配置シミュレーション機能の開発<br>- ✓ AI/最適化アルゴリズムの検討 |
| **Level 6** | **省人化**<br>システムやツールの活用により、必要な人員が削減されている状態 | 配置案の自動生成。リーダーによる確認・微調整のみ。配置業務工数の大幅削減 | - ✓ 配置最適化システム<br>- ✓ 自動生成アルゴリズム<br>- ✓ シミュレーション機能<br>- 少人数管理体制<br>- ✓ システム運用スキル | - ✓ 完全自動化・AI化の検討<br>- ✓ リアルタイム最適配置の実現<br>- ✓ 組織変革・人材育成 |
| **Level 7** | **人排除orAI化（ありたい姿）**<br>人の介在が最小限となり、AI等による自動判断・自動実行が実現している状態 | 需要予測・スキルマッチング・配置最適化の自動実行。人による例外対応のみ | - ✓ AI配置最適化システム<br>- ✓ 需要予測AI<br>- ✓ リアルタイム作業進捗連携基盤<br>- ✓ 自動実行インフラ | - ✓ システムの継続的学習・改善<br>- ✓ 新技術の評価・導入 |
