# 業務成熟度マトリクス（デジボ）

## 評価対象業務

**業務**: 製造ライン人員配置管理業務

**業務KPI（全レベル共通）:**
- 配置計画作成時間（分）
- 配置変更対応時間（分）
- スキルマッチング精度（%）

---

## 業務成熟度マトリクス

※ ✓ = 当部門担当

| レベル | 状態定義 | 業務の状態 | レベル到達の非定常施策・投資 | レベル維持の定常施策・投資 |
|:---:|:---|:---|:---|:---|
| **Level 0** | **プロセス未体系化** | リーダーの経験と勘による口頭指示。配置記録なし | （Level 0は記載なし） | - リーダーの経験と知識<br>- 口頭コミュニケーション |
| **Level 1** | **プロセス体系化** | 配置手順の文書化。配置図の作成。配置ルールの明文化 | - ✓ 業務プロセスの可視化・文書化<br>- ✓ 配置ルールの暗黙知の抽出 | - 配置手順書<br>- 配置図テンプレート（紙）<br>- 配置ルール文書 |
| **Level 2** | **手順標準化** | SOPに基づく配置計画。作業者リストとスキルの電子管理。配置状況の物理的可視化 | - ✓ SOP作成<br>- 作業者スキル一覧表の整備 | - 配置SOP<br>- 電子管理表（Excel等）<br>- ホワイトボード+ネームプレート等<br>- スキル一覧表 |
| **Level 3** | **手順遵守** | SOP通りの配置の徹底。配置変更時の必須記録・報告。定期的な実績振り返り | - 手順遵守の仕組み構築<br>- 逸脱時の記録・報告ルール整備 | - 配置変更記録フォーマット<br>- 定期レビュー会議体<br>- 逸脱管理台帳 |
| **Level 4** | **可視化** | 配置変更の即時記録。スキル情報の即時更新。配置状況の常時最新可視化 | - ✓ デジタル配置管理ツール導入<br>- ✓ リアルタイム記録の仕組み構築 | - ✓ **<font color="red">デジタル配置管理ツール（デジボ等）</font>**<br>- ✓ 基幹システム（ロジタン等）<br>- 表示用モニター<br>- 作業者マスターデータ<br>- ネットワーク環境・PC |
| **Level 5** | **最適化** | 蓄積データに基づく配置パターン最適化。計画的な多能工化推進。KPI管理による継続改善 | - ✓ データ分析体制の構築<br>- ✓ 改善PDCAサイクルの確立 | - ✓ データ分析ツール<br>- ✓ データ分析スキル・知識<br>- 改善会議体<br>- ✓ KPIダッシュボード<br>- 多能工化推進計画 |
| **Level 6** | **省人化** | 配置案の自動生成。リーダーによる確認・微調整のみ。配置業務工数の大幅削減 | - ✓ 配置シミュレーション機能の開発<br>- ✓ AI/最適化アルゴリズムの検討 | - ✓ 配置最適化システム<br>- ✓ 自動生成アルゴリズム<br>- ✓ シミュレーション機能<br>- 少人数管理体制 |
| **Level 7** | **人排除orAI化（ありたい姿）** | 需要予測・スキルマッチング・配置最適化の自動実行。人による例外対応のみ | - ✓ 完全自動化・AI化の検討<br>- ✓ リアルタイム最適配置の実現<br>- ✓ 組織変革・人材育成 | - ✓ AI配置最適化システム<br>- ✓ 需要予測AI<br>- ✓ リアルタイム作業進捗連携基盤<br>- ✓ 自動実行インフラ |
