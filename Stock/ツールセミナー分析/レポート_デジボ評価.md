# デジボ 評価レポート

## 1. ツール概要

**製品:** ロジタン デジボ（クラウド型デジタルホワイトボード）  
**目的:** ネームプレート準備・配置管理のデジタル化  
**料金:** ロジタン基本料金（年間24,000円〜）+ アドオン料金

## 2. 主な機能と効果

| 機能 | 効果 |
| :--- | :--- |
| ネームプレート自動生成 | 200名規模で6時間→ほぼゼロ（E：排除） |
| スキル自動更新 | 手動更新作業を排除（E：排除） |
| 配置変更時の自動計測 | 配置管理と時間計測を結合（C：結合） |
| パターン機能 | 配置変更を簡素化（S：簡素化） |

## 3. 制約事項

- ロジタン単独利用不可（アドオン必須）
- BI機能なし（Excel等で分析）
- マスターデータ手動入力必要
- 人員配置専用（物品管理等は不可）

## 4. 有用性評価

| 評価項目 | 判定 | 理由 |
| :--- | :--- | :--- |
| **適用現場** | △ | 現場での管理が不要・常駐デスクワーク前提 |
| **効果** | △ | アナログ→デジタル置き換えのみ |
| **データ連携** | × | I/O連携の自由度不明 |
| **汎用性** | × | 人員配置以外に使えない |
| **差別化** | × | 他ツールとの優位性不明 |

## 5. 総合判断

**判定: 導入推奨しない**

### 理由

1. **置き換え効果のみ:** 業務改善ではなく単なるデジタル化
2. **データ連携不明:** 人事DBとの連携・外部出力の可否が不明瞭
3. **汎用性欠如:** 人員配置専用で応用が効かない
4. **差別化不足:** より汎用的なツールで代替可能

### 導入前の必須確認事項

- 人事DBからのデータ自動連携は可能か？
- ツール外へのリアルタイムデータ出力は可能か？
- 他の汎用ツールとの比較検討を実施したか？
